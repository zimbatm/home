# backlog/

Work queue for the home fleet — `/grind` consumes it. One file per item
(`<area>-<slug>.md`: what/why/how-much/blockers); delete when done.

`ops-*` items need a human (`kin deploy`, `kin set` with secrets) — triage
marks them "needs-human" instead of picking.

`tried/` and `wontfix/` stop retreading.

Cross-repo: file `../kin/backlog/` etc. when the fix belongs upstream.
